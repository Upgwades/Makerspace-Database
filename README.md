# Makerspace-Database
A Database solution for the Florida Poly Makerspace

This project is a php website that is designed to facilitate the operations of a 3d printing labs with several printers.

This project heaviliy relies on deployment in AWS Elastic Beanstalk with a backend mysql database and file hosting on AWS S3 services.

Features include: user account creation, project requesting/managing, useful resources etc.

The aims is that this project could be used as a template for printing labs cut wasted time out of the print request process and get request delivered to their requestors.
